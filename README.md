# 📊 Customer Purchase Data Analysis (Pandas Exercise)

This project is a **Pandas data analysis exercise** using a simulated customer purchase dataset.
The notebook analyzes customer information such as **age, profession, purchase amount, credit card details, and location** to answer several business questions.

The goal of this project is to practice **data analysis using Python and Pandas** by exploring and extracting insights from a dataset with **30,000 customer records**.

---

# 📁 Project File

`Cust_Purch_Data_Exercise.ipynb`

This Jupyter Notebook contains:

* Data loading using **Pandas**
* Data exploration and inspection
* Data filtering and aggregation
* Answering business questions using Pandas functions

The dataset used in this notebook is:

`Cust_Purch_FakeData.csv`

This dataset simulates customer purchase data collected from an e-commerce or retail system.

---

# 🧠 Topics Covered

This notebook demonstrates how to:

* Import and use **Pandas**
* Load datasets using `read_csv()`
* Explore data using:

  * `head()`
  * `info()`
  * `describe()`
* Perform **data analysis queries**
* Use functions such as:

  * `value_counts()`
  * `mean()`
  * `max()`
  * `min()`
  * `sum()`
  * `count()`
* Filter data based on conditions
* Extract useful business insights

---

# 📊 Example Analysis Questions

The notebook answers several analytical questions such as:

* How many entries and columns exist in the dataset?
* What are the **maximum, minimum, and average ages** of customers?
* What are the **three most common customer names**?
* Which customers share the **same phone number**?
* How many customers work as **Structural Engineers**?
* How many **male Structural Engineers** exist?
* How many **female Structural Engineers from Alberta (AB)** exist?
* What are the **maximum, minimum, and average spending values**?
* Which customers **did not spend anything**?
* Which customers spent **100 CAD or more**?
* How many emails are associated with a **specific credit card number**?
* How many **credit cards expire in 2019**?

These questions simulate **real business analytics tasks**.
